Social Scope: Sentiment Analysis Feature

Objective:
To evaluate whether the comments on an Instagram reel effectively respond to the question posed by the content creator at the end of the reel. This analysis will provide insights into audience engagement and the relevance of the content to the viewers.

Target Reel:

Instagram Reel: Did the comments answer the question?

Approach:

  • Comment Extraction:
    • Utilize the Instaloader Python package to extract all comments from the reel.
    • Run a scheduled batch job every 12 hours to extract the latest comments and update the database.
  • Data Analysis:
    • Compile the extracted comments along with other relevant post data.
    • Utilize the latest language model (LLM, LAMMA 3.1) to generate a summary or "gist" of the overall sentiment and engagement reflected in the comments.
    • Evaluate whether the audience effectively addressed the question posed in the reel.
  • Post-Analysis Reporting:
    • Automatically generate and send a detailed analysis report to the Instagram account owner via email.
    • The report will include insights on audience engagement, suggestions for future content creation, and recommendations for similar topics that align with audience interests.

Outcome:

By leveraging AI-powered sentiment analysis, content creators can gain a deeper understanding of how their audience interacts with their posts and receive actionable insights to improve future content.
